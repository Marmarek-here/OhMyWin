# OhMyWin
OhMyWin is a Zsh theme made to provide the looks of cmd to zsh for either new-Linux (or macOS) users that came from Windows or simple customization for more advanced users.

# Installation
To install, you need:
- Zsh
- Oh-My-Zsh
- Git

To install, type into your console:
```git clone https://github.com/Marmarek-here/OhMyWin
cd OhMyWin
./install.sh --full --yes
```

# Flags
    --minimal
        Theme only.
        Also has a sub-option:
            --yes
                Zero interaction, automated.
    
    --full
        Everything installed + copyright banner (like on Windows) enabled. (Not needed if you don't need to see the copyright and your OS's name)

    --custom
        Ask for everything. (Can be set up for yourself (doesn't have colors))
